# GoCompress

Compressez vos images et PDF gratuitement.


## Ce qu'il fait

- **Images** : PNG, JPG, JPEG, WebP, BMP, TIFF
- **PDF** : réduction de taille intelligente
- Tout se fait dans votre navigateur (rien ne part sur un serveur)
- Résultats affichés avant/après
- Téléchargement en un clic

## Comment ça marche ?

1. Cliquez sur "Compresser les images" ou "Compresser les PDF"
2. Choisissez votre fichier
3. La compression commence automatiquement
4. Voyez la différence de taille
5. Téléchargez le fichier compressé

## Technologies utilisées

- HTML / CSS / JavaScript (vanilla)
- browser-image-compression (pour les images)
- pdf-lib (pour les PDF)
- Vercel (hébergement)
