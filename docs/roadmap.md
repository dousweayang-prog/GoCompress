# Roadmap GoCompress

## Version 1.0 ✅ (Terminé)
- [x] Interface minimaliste
- [x] Compression d'images (PNG, JPG, JPEG, WebP, BMP, TIFF)
- [x] Compression de PDF
- [x] Modal avec progression
- [x] Affichage des résultats avant/après
- [x] Suppression automatique des fichiers en mémoire
- [x] Responsive (mobile, tablette, desktop)
- [x] Déploiement sur Vercel

---

## Version 1.1 (À venir)
- [ ] Ajout d'un glisser-déposer plus intuitif
- [ ] Barre de progression plus détaillée
- [ ] Compression en lot (plusieurs fichiers à la fois)
- [ ] Amélioration de la compression PDF
- [ ] Mode sombre/clair (toggle)

---

## Version 2.0 (Futur)
- [ ] Compression audio (MP3, WAV, AAC)
- [ ] Compression vidéo (MP4, WebM, AVI)
- [ ] Compression ZIP (archives)
- [ ] Conversion de fichiers
  - [ ] Image → PNG, JPG, WebP
  - [ ] PDF → Image
  - [ ] Audio → MP3
- [ ] Multilingue (Français, Anglais, Espagnol)
- [ ] Progressive Web App (PWA)
- [ ] Tableau de bord des statistiques

---

## Version 3.0 (Long terme)
- [ ] API publique (pour intégration tierce)
- [ ] Authentification (comptes utilisateurs)
- [ ] Historique des compressions
- [ ] Partage de fichiers compressés
- [ ] Compression avec IA (optimisation intelligente)
